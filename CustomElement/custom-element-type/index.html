custom-element-type

Autonomous custom elements(自主定义元素)

Autonomous custom elements 是独立的元素，它不继承其他内建的HTML元素。可以直接把它们写成HTML标签的形式，在页面上使用。
可以直接使用 custom-element-type 标签，或者 document.createElement("custom-element-type")。

Customized built-in elements(内置定制元素)

Customized built-in elements 继承自基本的HTML元素。在创建时，必须指定所需扩展的元素，使用时需先写出基本的元素标签，并通过 is 属性指定custom element的名称。
例如 p is="built-in-element", 或者 document.createElement("p", { is: "built-in-element" })。
